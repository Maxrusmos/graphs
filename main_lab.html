About

The program is used to draw graphs and solve some problems. You can draw graphs on the adjacency matrix by adding certain values to it:
1) 1 or 0 for unloaded and undirected graph. For example:

2) From 0 to 100 for loaded and undirected graph. For example:

3) From 0 to 100 for loaded and directed graph. For examle:

4) 0 or 1 for unloaded and directed directed graph. For example:

5) you can draw a multigraph. For example:

You can change the location of vertices by dragging them with the mouse over the field. In the context menu for the graph, you can change the names of vertices, delete vertices, make a loop (first select the vertex).

There is a functional for saving to a file in the form of adjacency matrices, incidence, list of vertices. You can read from a file in the formats of adjacency matrix, incident matrix, vertex list. There are functions to return up to 10 steps back and return 10 steps forward.
